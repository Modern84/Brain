# Prompts — Textbausteine für Claude-Sessions

Vorgefertigte Prompts für claude.ai (und teilweise Claude Code). Copy & Paste, einsetzen wo nötig.

## Übersicht

| Prompt | Zweck | Wo einsetzen |
|---|---|---|
| `Session-Starter.md` | Neuen claude.ai-Tab auf aktuellen Stand bringen | Anfang eines neuen Tabs |
| `Text-Check-LRS.md` | Text still korrigieren ohne Kommentare | Wenn Sebastian einen Entwurf schickt |
| `WEC-Montag-Session.md` | Montags-Session mit Reiner vorbereiten | Sonntagabend vor Reiner-Treffen |
| `Mail-Ton-Varianten.md` | Korrekte Tonalität je Empfänger erkennen | Bei allen Mail-Entwürfen |

## Handover-Prompts

Zusätzlich in [[04 Ressourcen/Templates/Claude-Handover/]] — Prompts für Übergabe zwischen **claude.ai** und **Claude Code** (parallel-Betrieb).

## Prinzipien

- **Kurz** — Prompts > 500 Zeichen werden selten verwendet
- **Konkret** — eindeutiger Auftrag, kein Romantext
- **Output-Format vorgeben** — Markdown, nur korrigierter Text, etc.
- **Sebastian-spezifisch** — LRS, Reiner-Ton, WEC-Kontext eingebaut

# Claude for Chrome — Browser-Extension

## Was es ist

Browser-Extension die Claude direkt im Chrome-Kontext verfügbar macht.

- Text auf einer Seite markieren → Rechtsklick-Kontextmenü oder Mini-Popup
- Aktionen direkt auslösen: Zusammenfassen, Erklären, Übersetzen
- Kein manuelles Kopieren nach claude.ai nötig

## Installation

chrome.google.com/webstore → „Claude for Chrome" suchen

## Wann nützlich

- Recherche: Artikel sofort zusammenfassen ohne Tab-Wechsel
- Übersetzung: Fremdsprachige Texte on-the-fly
- Erklärungen: Technische Dokumentation direkt erläutern lassen

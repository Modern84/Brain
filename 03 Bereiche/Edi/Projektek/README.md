# Projektek

Itt vannak a konkrét projektek, amik **Mo fejlesztéseiből magyar vállalkozásba csatornázódnak**.

## Ajánlott struktúra egy projekthez

```
ProjektNeve/
├── áttekintés.md      — mi ez, mi a cél
├── státusz.md         — hol tart most
├── kapcsolatok.md     — kikkel dolgozunk ezen
└── dokumentumok/      — szerződések, specifikációk, stb.
```

## Hogyan kezdjünk el egy projektet?

1. Kérd meg Mo-t, hogy meséljen egyről — nyugodtan németül, Claude fordít.
2. Claude segít összefoglalni magyarul, mi az, és mi lenne a magyar cég szerepe.
3. Új mappa → első `áttekintés.md` → onnan építjük.

# Pályázatok és támogatások

Itt gyűjtjük a releváns pályázati információkat.

## Ajánlott struktúra

```
Pályázatok/
├── Figyelt/           — amire készülünk
├── Aktív/             — amire éppen pályázunk
├── Megnyert/          — amit sikerült (ünneplünk!)
└── Elutasított/       — amit nem adott elő magát (rendben — tanulás)
```

## Első lépés javaslat

Hozz létre egy `magyar_startup_palyazatok_2026.md` fájlt, és kezdd el oda rakni, amit találsz.

Szempontok:
- Cégforma (Kft., Bt., egyéni vállalkozó — mi passzol)
- Tematika (innováció, digitalizáció, export, stb.)
- Beadási határidő
- Önerő-igény
- Elbírálási idő

## Források, ahol érdemes keresni

- **Széchenyi Programiroda** — magyar állami pályázatok
- **EU Funding & Tenders Portal** — európai források
- **NKFI Hivatal** — kutatás-fejlesztési támogatások
- **Magyar Fejlesztési Bank** — startup-finanszírozás

## Claude segíthet

- Összefoglalni egy pályázati kiírást magyarul, egyszerű nyelven
- Ellenőrizni, megfelelünk-e a kritériumoknak
- Megfogalmazni a pályázati szöveget
- Határidőket naptárba rakni

Csak írj: *„Claude, találtam egy pályázatot, segítsél átlátni."*

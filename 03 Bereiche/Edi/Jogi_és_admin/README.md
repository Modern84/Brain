# Jogi és adminisztratív alap

Itt minden, ami a cég jogi és adminisztratív lábához tartozik.

## Ajánlott struktúra

```
Jogi_és_admin/
├── Könyvelés/         — könyvelő, számlák, adó
├── Szerződések/       — partnerszerződések, mintadokumentumok
├── Hatóság/           — NAV, cégbíróság, egyéb
└── Bank/              — számla, kártya, online banking
```

## Első lépés javaslat

Hozz létre egy `könyvelő_kiválasztás.md` fájlt, és kezdd el oda gyűjteni:
- Kikkel beszéltél
- Mit kértek havi díjban
- Milyen tapasztalatuk van startup / innovációs cégekkel
- Beszélnek-e angolul vagy németül (ha Mo-nak is kell kapcsolat)

## Mit érdemes minél hamarabb rendezni?

1. **Könyvelő** — stabil kapcsolat, aki érti az üzleti logikádat
2. **Céges bankszámla** — ha még nincs
3. **Alapszerződés-minták** — NDA, partnerségi, megbízási
4. **Hatósági regisztrációk** — NAV, cégbíróság rendben

## Claude segíthet

- Egyszerűsíteni a jogi szövegeket magyarul
- Összehasonlítani ajánlatokat
- Emlékeztetni határidőkre (adóbevallás, beszámoló)
- Segíteni e-mailt megfogalmazni hatóságnak, partnernek

**Fontos:** Claude nem ügyvéd. Ha valami komoly jogi kérdés, szakértő kell. Claude segít eldönteni, mikor éri meg szakértőhöz fordulni.
